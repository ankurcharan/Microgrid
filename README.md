# Microgrid

**Steps**
- Run Ganache
- Enter you RPC Server in config.json
```
> $npm install --save
> $node deploy.js
```
- Will give contract address, on terminal, copy & paste in config.json
```
> $node simulation.js - (to run simulation)
> $node server.js - (to generate object for UI)
```
